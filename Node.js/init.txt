REPL Read Eval Print Loop
사용자가 command를 입력하면 시스템이 값을 반환하는 환경


Read    – 유저의 값을 입력 받아 JavaScript 데이터 구조로 메모리에 저장합니다.
Eval    – 데이터를 처리(Evaluate) 합니다.
Print   – 결과값을 출력합니다.
Loop    – Read, Eval, Print 를 유저가 Ctrl+C를 두번 눌러 종료할때까지 반복합니다.
